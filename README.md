# Supply Chain Logistics Dashboard 📊

## Project Overview
This project analyzes a massive supply chain dataset containing **113,000+ rows** of logistics data. The goal was to clean the messy data, analyze order behavior, and build an interactive reporting tool for management.

## Tools Used
- **Advanced Excel** (Pivot Tables, Charts, Multi-condition Logic)
- **Power Query** (Data Cleaning, Custom/Conditional Columns)

## Key Insights
- Successfully categorized order completion statuses into 'Completed' and 'Pending'.
- Built an interactive **Slicer-driven Pie Chart** allowing management to filter data by handling equipment availability instantly.
